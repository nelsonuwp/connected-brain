# 60-decisions  ·  Records

Decision records. One file per non-trivial decision. ADR style.

**Contains:**
- What was decided (one sentence)
- Context and options considered
- Why alternatives were rejected
- Consequences and follow-on actions

**Rules:**
- Log when: >30 minutes of deliberation, affects multiple people, or will be questioned later
- Inject when an LLM asks "why are we doing it this way?"
- Never modify a logged decision — create a new record that supersedes it
- Naming: DR-001.md or 2026-02-21-decision-name.md
