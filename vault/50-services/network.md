## Service Description

The Network service designs, implements, optimizes, and manages resilient and scalable network infrastructure for internal operations and customer-facing environments, enabling customers and internal teams to connect and access their assets reliably and securely. It delivers network provisioning, access control, performance monitoring, compliance evidence, and IP address management so that customers experience consistent connectivity and internal teams can enforce security boundaries across the environments they manage.

## User Needs

- As a **Customer**, I need a managed network so that my assets are accessible as required by my business.

- As a **Customer**, I need to attach and detach assets from the network so that they can be accessed or isolated remotely as my environment changes.

- As a **Customer**, I need IP addresses allocated to my environment so that I can connect my devices to the internet.

- As an **Infrastructure team member**, I need to restrict access to specific parts of the network so that I can protect the integrity of my customers' environments.

- As an **Auditor**, I need compliance evidence from the network service so that I can verify that regulatory and security assertions have been met.
