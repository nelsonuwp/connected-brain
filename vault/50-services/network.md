## Service Description

The Network service provides resilient, scalable network infrastructure, enabling customers and internal teams to access, connect, and secure their assets. It delivers IP address management, network segmentation, access control, and performance monitoring so that customers can operate their business with confidence, knowing their connectivity is reliable, protected, and professionally managed.

*As a [Persona], I need [Capability] so that [Outcome].*

**As a Customer, I need my network to be consistently available so that my assets can be accessed as required.**

**As a Customer, I need to attach and detach assets from the network so that they can be accessed or isolated remotely as my needs change.**

**As a Customer, I need access to parts of the network to be restricted so that the integrity of my business is protected.**

**As a Customer, I need IP addresses provisioned so that I can connect my devices to the internet.**

**As an Auditor, I need evidence of compliance with network standards so that I can confirm required obligations have been met.**
