# 90-meeting-notes  ·  Comms

Meeting notes. One file per meeting.

**Naming:** YYYY-MM-DD-meeting-name.md

**Contains:**
- Attendees, raw notes, decisions made, action items with owners

**Rules:**
- Create from _templates/meeting-note.md before the meeting
- For 1:1s: also open the person note from 40-people beforehand
- After the meeting: run _prompts/meeting-summary.md on raw notes
- Route outputs: actions → 40-people, decisions → 60-decisions,
  context → 20-context, open loops → open-loops.md
- The note stays here as a record — not a living document
