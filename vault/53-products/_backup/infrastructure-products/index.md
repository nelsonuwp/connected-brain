# Infrastructure Products

Aptum delivers infrastructure across six commodity types: colocation, servers (bare metal), connectivity, Aptum IaaS (virtual private cloud and dedicated cloud), private cloud on VMware or Proxmox, and public cloud via hyperscaler partnerships. Each commodity is a standalone product. Managed services, engagement tiers, and addons stack on top of any commodity and are documented separately in the Managed Services section.

Every infrastructure product includes a Fundamental guarantee: the non-negotiable floor Aptum delivers regardless of engagement tier or addon selection. If a customer buys nothing else, they still get the Fundamental. The Fundamental varies by product and is documented on each product page.

The tenancy model differs across products. Colocation and connectivity are physical resources dedicated to a single customer. Bare metal servers are single-tenant by definition. Aptum IaaS Shared Cluster (VPC) uses shared physical hosts with logical isolation via VLANs. Aptum IaaS Dedicated Cloud and Private Cloud use dedicated physical hosts. Public cloud follows the hyperscaler tenancy model.

Aptum operates data centers across 26 locations in 21 cities, spanning North America and the UK. Primary sites include Herndon VA (1,339 services), Toronto ON (1,203 services), Portsmouth UK (1,091 services), Atlanta GA (591 services), Miami FL (570 services), and Los Angeles CA (551 services). The Toronto Pullman data center is the primary site for Aptum IaaS (VPC and Dedicated Cloud) and carries SOC 2 Type II certification.
