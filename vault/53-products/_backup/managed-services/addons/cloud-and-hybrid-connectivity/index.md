# Cloud and Hybrid Connectivity

Two products for customers running workloads across Aptum infrastructure and hyperscaler environments. Both apply to any customer with a hybrid or multi-cloud architecture.

## Aptum Hybrid Interconnects

Dedicated private circuits connecting Aptum infrastructure to hyperscalers, bypassing the public internet for security, performance, and billing predictability. Azure ExpressRoute, AWS Direct Connect, and GCP Partner Interconnect are all supported. At Reactive tier, the physical circuit is provisioned and monitored automatically, with circuit health and uptime reporting. At Proactive tier, Managed Cloud manages logical configuration, ongoing circuit monitoring, failover setup and testing, and change management. Owners: Networking (physical circuit); Managed Cloud (logical configuration and management). Cost: Azure ExpressRoute 50Mbps approximately $75 CAD per month; 200Mbps approximately $200 CAD per month. AWS Direct Connect hosted connection approximately $30 to $120 CAD per month for the port fee, with data transfer charges separate.

## Aptum FinOps

Cloud spend visibility, anomaly detection, budget governance, and active optimization for public cloud and Aptum IaaS. This is active optimization — not billing support, which is a Fundamental for public cloud customers. At Reactive tier, Aptum Portal cost dashboards provide automated spend reporting, budget alerts, anomaly detection, and utilization tracking; the customer has full dashboard access. At Proactive tier, Managed Cloud conducts rightsizing analysis, develops reserved instance strategy, designs governance frameworks, and runs monthly optimization reviews. Owners: Managed Cloud (analysis and recommendations); Aptum Portal (visibility tooling). Cost: included in Foundation and Aptum Portal at Reactive tier.
