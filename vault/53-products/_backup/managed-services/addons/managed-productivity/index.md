# Managed Productivity

## Aptum Managed M365

Exchange Online, SharePoint, and Teams administration: user provisioning, security configuration, and license management. M365 SaaS layer only — this is not Azure infrastructure management (Azure is a separate product in the Infrastructure Products section). M365 licensing is customer-supplied; Aptum charges for the management layer. At Reactive tier, Managed Cloud manages user provisioning, security policies, license optimization, and governance. Status: live. Owner: Managed Cloud. Commercial model: M365 licensing is the customer's cost. Management time priced as flat monthly fee or per-seat rate (TBD).
