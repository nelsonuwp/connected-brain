# 00-daily  ·  Time

Daily notes and weekly reviews.

**File naming:**
- Daily:  YYYY-MM-DD.md  (e.g., 2026-02-21.md)
- Weekly: YYYY-Www.md    (e.g., 2026-W08.md)

**Contains:**
- Today's schedule (calendar events with links to pre-created meeting notes)
- Carry-over tasks (via Tasks plugin query)
- Cognitive load dashboard (active initiatives count, delegation count, open loop count)
- End-of-day captures

**Rules:**
- Generated from _templates/daily-briefing.md — don't create from scratch
- Daily notes are operational, not archival — don't store anything here that needs to survive
- Anything worth keeping gets promoted to the right folder during triage
