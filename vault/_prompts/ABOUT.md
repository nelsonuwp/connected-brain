# _prompts  ·  System

Reusable LLM prompt templates. Copy, fill the variables, send.

- meeting-summary.md, one-on-one-prep.md

**Rules:**

- Every prompt uses the standard header: MODE / CONTEXT BLOCKS / TASK / OUTPUT FORMAT
- If you write the same prompt structure twice → add it here before the third time
