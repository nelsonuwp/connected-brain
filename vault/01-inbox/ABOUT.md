# 01-inbox  ·  Time

Raw, unprocessed captures. First stop for everything.

**Contains:**
- Thoughts captured on the fly
- Pasted Slack/Teams/email threads that need processing
- Shower thoughts, meeting follow-ups, anything not yet routed

**Rules:**
- Default landing folder for all new Obsidian notes
- 48-hour maximum shelf life — everything gets promoted or deleted
- Never send content from here directly to an LLM — abstraction level is too mixed
- Triage every morning: route to 10-thinking, 30-initiatives, 20-context, or delete
