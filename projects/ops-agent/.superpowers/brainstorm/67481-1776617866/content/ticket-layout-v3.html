Single-column layouts — sidebar removed

All three options assume no right sidebar. The left-nav stub shows how these will sit inside the future org dashboard shell. Related issues and AI actions need new homes in each approach.

◎ Each mockup shows the left-nav stub that your future shell will provide — the content area is what changes.
A

Inline AI panel

Header → AI panel → description → thread. Related issues become horizontal chips above the AI panel. One output area, no duplication.

Pros
  • Single scroll
  • No sidebar needed
  • Simple to implement
Cons
  • AI above description
  • Output pushes content down
  • Related chips feel secondary

Cons

  • Dense on small screens
