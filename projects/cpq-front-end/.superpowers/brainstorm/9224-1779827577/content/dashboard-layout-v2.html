Next question: What drives the "cost" number?

We have two sources for cost data — which should this page use?

A

Actual financial data

Pull from FinancialReporting.Profit_Analysis and Profit_Cost tables — real billed costs as recorded by finance. Most accurate, reflects what actually happened.

B

CPQ-calculated model

Same formula used in Renewals: MRC minus overhead (power, DC ops, SG&A) minus HW amortization. Forward-looking and consistent with quotes. Good for "what should margin be".

C

Both — actual vs. modeled

Show the CPQ model as the baseline, flag where actual financial data diverges. Powerful for finding where reality doesn't match expectations — but more complex to build.
